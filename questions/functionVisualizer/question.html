The question below was designed to showcase the different features of pl-interactive-visualizer. This element creates a 3-dimensional plot of a function that users can rotate, zoom, and place points on. This is a decorative element, and placed points are not saved and graded in the current version. For grading, the element should be combined with with `pl-number-input` or other elements where students enter and submit numeric coordinates.

See the other example questions for more realistic question setups and randomization examples.

Element Demo

Below is an example of a function visualization for $3x+x^2y+|y|$, written as the function expression 3*x+x**2*y+abs(y). The parameter place is set to true to allow students to select up to 4 points that are displayed next to the function. These points are not saved or graded unless they are manually entered into a different element, such as pl-number-input.
